# daimaiqr
# 抢票助手
将大麦要抢得场次复制 使用此工具 转换为二维码,大麦app扫码进入,推荐安卓,ios要选观影人.
 

[直接使用](https://daimaiqr.vercel.app/)

提前十分左右扫码,到点直接点击确认!提交订单就行

此工具只提供bp链接的生成,想找全自动的不要想了. 

全自动的全都是收费的,如果你有时间,善于专研,可以自己尝试下.

只提供链接的转换,进抢票群 代抢 出票 +v:

同时也回收茅台 数码.
